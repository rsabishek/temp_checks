Annotations:

The configuration begins by configuring the pingfederate provider using the provider block. Replace the values of base_url, username, and password with the appropriate credentials for your PingFederate instance. Set skip_cert_verification to false to enforce certificate verification.

The resource block defines the creation of an OAuth client instance using the pingfederate_oauth_client resource type.

client_id specifies a unique identifier for the OAuth client.

display_name is a human-readable name for the OAuth client.

redirect_uris is a list of URIs to which the authorization callback will be sent.

allowed_grant_types lists the grant types that the OAuth client is allowed to use.

allowed_scopes lists the scopes that the OAuth client can request during authorization.

enabled determines whether the client is enabled or not.
